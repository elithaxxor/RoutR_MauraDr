# Complementary Feature Proposal

To further strengthen RoutR_MauraDr, the following ideas could be considered:

1. **Automated Certificate Renewal Suggestions**
   - Notify administrators when discovered certificates are nearing expiration.
   - Provide recommended `openssl` commands for generating new CSRs.

2. **Baseline Configuration Templates**
   - Offer downloadable hardening templates for common routers.
   - Compare current settings against these templates to highlight gaps.

3. **Graph-Based Data Explorer**
   - Extend the D3.js topology to allow filtering by protocol, port or vendor.
   - Support overlaying custom labels and notes for each node.

4. **Risk Scoring Dashboard**
   - Apply simple machine learning to weigh findings and surface critical issues.
   - Display per-device risk scores in the web interface.
